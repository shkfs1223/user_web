# 双碳管理系统

1. ## 检查开发环境
   ```batchfile
   node -v //node版本大于16
   ```
2. ## 下载pnpm
   ```batchfile
   npm i pnpm -g
   ```
3. ## 配置淘宝源
   ```batchfile
   pnpm config set registry https://registry.npmmirror.com/
   ```
4. ## 安装依赖
   ```batchfile
   pnpm i
   ```
5. ## 运行项目
   ```batchfile
   pnpm run dev
   ```
6. ## 项目打包
   ```batchfile
   pnpm run build:prod
   ```
